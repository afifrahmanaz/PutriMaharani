# Portfolio Putri Rahmadani

Portfolio website untuk Putri Rahmadani - Graphic Designer & Sales/Marketing Professional

## 🚀 Cara Menjalankan

```bash
# Install dependencies
npm install

# Jalankan development server
npm run dev
```

Server akan berjalan di `http://localhost:3000` (atau port lain jika 3000 sedang digunakan)

## 📁 Struktur Project

- `/app` - Next.js App Router pages
  - `/design` - Halaman portfolio design & landing page
  - `/social-media` - Halaman portfolio social media & graphic content
  - `/video` - Halaman portfolio video content
- `/components` - Reusable React components
- `/public` - Static assets (gambar, CV, dll)

## 🎨 Fitur

- ✅ Responsive design untuk mobile dan desktop
- ✅ Modern glassmorphism UI
- ✅ Smooth animations dan transitions
- ✅ Portfolio showcase untuk berbagai kategori
- ✅ Social media integration

## 📦 Teknologi

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide React Icons

## 📝 Catatan

**File yang perlu ditambahkan:**

1. **Profile Image**: Tambahkan foto profil di `/public/profilputri/PUTRI.jpg`
